# Cafè Sales Analytics & Seasonal Marketing Strategy

Solving one of the biggest marketing problems for coffee shops, customer preferences based on seasonality and trends. This project uses data to explore insights and provide marketing recommendations for and NYC coffee shop.

## Project Overview

This project analyzes transactional sales data from a New York City coffee shop to understand how customer purchasing behavior changes across seasons and how those patterns can inform marketing decisions.

The objective was to move beyond descriptive reporting and develop a predictive approach that helps businesses identify which products are likely to perform best before each season begins. The analysis combines exploratory data analysis with machine learning to translate sales data into practical business recommendations.

---

## Business Question

Retail businesses frequently promote products based on intuition rather than evidence. This project explores how data can support decisions such as:

* Which products generate the most revenue in each season?
* How should advertising strategies change throughout the year?
* Can future high-performing products be predicted using historical data?

---

## Methodology

The workflow followed a standard analytics pipeline:

* Data cleaning and feature engineering, including seasonal classification from transaction dates
* Exploratory analysis to identify revenue trends and product performance
* Predictive modeling using classification algorithms

Two models were evaluated:

* Logistic Regression (baseline model)
* Random Forest Classifier (final selected model)

The Random Forest model achieved an accuracy of **91.7%**, demonstrating strong performance in capturing seasonal purchasing patterns.

---

## Key Findings

The analysis revealed consistent seasonal behavior that can directly inform marketing strategy:

* **Spring:** Chai beverages and pastries showed strong performance, suggesting bundled promotions during slower hours.
* **Summer:** Demand shifted toward iced and brewed coffee products, supporting cold-drink focused advertising.
* **Winter:** Warm beverages dominated revenue, indicating opportunities for seasonal or limited-edition menu items.

Detailed analysis and recommendations are available in the project reports.
